Package: gratum.etl

[Groovy] Class LoadStatistic

    • gratum.etl.LoadStatistic

    • This object contains the statistics on how many items were processed by the Pipeline. The name of the Pipeline is contained in the name property. Things included in this object are number of rows loaded, number of rows rejected, categories and count of each row rejected, the total time spent processing the Pipeline, and the time each step took to process the rows.
