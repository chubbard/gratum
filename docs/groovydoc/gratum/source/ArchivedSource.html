Package: gratum.source

[Groovy] Class ArchivedSource

      • gratum.source.ArchivedSource

      • ArchivedSource supports other archive formats that the JDK doesn't support. Things like Zip64 (ie Enhanced_Deflate), sevenZ (7z), arj, etc. This uses the apache common-compress library under the hood so any format it can read should be available with this source. Each row holds the following keys:

        • filename - Filename of the archive file.
        • file - underlying archive java.io.File that this came from.
        • entry - org.apache.commons.compress.archivers.ArchiveEntry representing a file within the archive
        • stream - The java.io.InputStream of this entry in the archive
        Examples: For Zip, Zip64, etc: ArchivedSource.unzip( file ).into() ... .go() For 7z: ArchivedSource.sevenZ( file ).into() ... .go() For auto-detect format: ArchivedSource.unarchive( file ).into() ... .go() Specific format: ArchivedSource.unarchive( file ).format( ArchiveStreamFactory.ARJ ).into() ... .go()
